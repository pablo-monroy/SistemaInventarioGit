Descripción
Programa desarrollado en lenguaje C que permite registrar, mostrar, modificar y eliminar equipos informáticos, este programa esta creado en Git con control de versiones y ramas

Funcionalidades
Registrar: Permite registrar un equipo con código alfanumérico , nombre, marca del equipo, precio y estado del mismo
Mostrar: Muestra los equipos registrados en el programa
modificar Equipo: Permite modificar los datos de un equipo ya registrado
Eliminar equipo: Permite buscar y eliminar un equipo con el código
Menú: Permite seleccionar que acción hacer e interactuar con las funciones

Tecnologías utilizadas
Lenguaje C: es el lenguaje utilizado para el programa
Librerías estándar: stdio.h, stdlib.h, string.h y windows.h contienen las etiquetas necesarias para el código
Git y GitHub: permiten gestionar el control de versiones
